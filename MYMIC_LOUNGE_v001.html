# 기획서: 2D 멀티유저 가상 라운지 & 실시간 오디오 노래방 웹 앱

## 1. 개요 및 목적
- 본 프로젝트는 사용자가 가상 공간에 개별 아바타로 참여해, 마이크 볼륨 및 공간 이펙트(리버브, 에코)를 실시간 조절하며 함께 노래할 수 있는 트렌디하고 가벼운 멀티유저 플랫폼입니다.
- 현대적인 세련된 UI 스타일을 지향하며, 웹 브라우저 및 스마트폰 브라우저에서 별도 프로그램 설치 없이 즉시 실행되는 뛰어난 접근성을 목표로 합니다.

## 2. 핵심 기능 요구 사항

### ① 실시간 2D 멀티플레이어 공간 맵
- 누구나 별도 절차 없이 게스트로 즉시 진입할 수 있는 오픈 라운지 형태.
- 2D Canvas 맵 내에서 마우스 클릭 혹은 모바일 화면 터치를 통해 계정/유저당 하나의 원형 아바타가 목표 지점까지 부드럽게 이동(인터폴레이션).
- 맵 내부에는 가상 유저들(시뮬레이션 캐릭터)이 배치되어 멀티 유저 느낌을 제공해야 함.

### ② 마이크 장치 획득 및 통신 권한 (모바일/PC 공통)
- 앱 진입 혹은 마이크 버튼 활성화 시 기기 마이크 사용 권한 동의 팝업을 직관적으로 송출.
- 모바일(iOS Safari, Android Chrome)에서도 하드웨어 마이크 디바이스 권한을 획득해 즉각적으로 지연 없이 연동 지원.

### ③ 실시간 Web Audio API 탑재 이펙트 조절 기능
- **마이크 토글 버튼**: 토글 방식으로 클릭 시 활성화(초록/보라색 글로잉 맥동 효과), 다시 한 번 클릭 시 비활성화(붉은색 및 슬래시 아이콘) 처리.
- **리버브(Reverb)**: 가상 임펄스 응답 소스(Impulse Response)를 소프트웨어 연산으로 생성하여 공간감(성당, 클럽 등) 잔향 깊이를 조절하는 볼륨 슬라이더 제공.
- **에코(Echo/Delay)**: 오디오 피드백 루프(Delay Node + Gain Feedback) 회로를 설계하여 반복 울림 횟수와 반사음 크기를 조절하는 슬라이더 제공.
- **조절창 제어**: 이펙트 아이콘 터치 시 팝업 슬라이더 콘솔이 보이며, 화면 내 조절창 이외 영역(바탕화면 맵 등)을 클릭하면 조절 창이 자동으로 닫히는 직관적인 UX.

## 3. 디자인 표준 (UI/UX)
- **테마**: 어두운 네온 사이버펑크(Dark Cyberpunk/Neon Blue & Purple) 및 반투명 글래스모피즘(Glassmorphism) 적용.
- **모바일 대응**: 모든 슬라이더와 아바타 터치 조작부는 스마트폰 화면 규격(모바일 뷰포트)에 맞게 반응형으로 최적화 설계.

## 4. 백엔드 및 오디오 전송 보안 설계 핵심 가이드
- **음원 저작권 보호**: 서버 S3 원본 음원 주소를 클라이언트에 노출하지 않고, 재생 시점에 백엔드에서 생성하는 5분 만료 한정 '임시 서명된 URL(AWS Pre-signed URL)' 구조 채택.
- **데이터 보안 보관**: 유저 액세스 토큰은 절대 로컬스토리지에 평문 저장하지 않으며 암호화 쿠키(httpOnly) 또는 보안 키체인(App Secure Storage)에 은닉 관리.
- **데이터 유효성 검증**: SQL Injection 및 XSS 방지를 위한 DTO(Data Transfer Object) 통제 및 API 유효성 필수 검증.
